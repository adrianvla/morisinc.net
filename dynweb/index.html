DynWeb

A very simple CMS

How It Works:

Lets assume DynWeb is installed on example.com (the script is included in index.html).

When a person goes to https://example.com/#{x}, the content from the file at https://example.com/a/{x}.html will be loaded into an element with the ID 'content'.

Making websites with this is very easy because you don't need to change every page of your website when you change something on the page and because you can now make pages by exporting content html files from Open Office.

How To set it up

  • Paste this in your <​body​> tag:
<div id="content"></div>
<script src="path/to/dynweb/here/replace/this"</script>
  • Now, make a new folder named "a"
  • You're done! It'll work. Now you can style it with css
