Tetris

I wanted to make Tetris

I wanted to make Tetris, so I've recreated the original tetris that you could play on the Soviet DVK-2M computer made by Alexey Pajitnov.

[btn link="https://projects.morisinc.net/dvk2m_tetris/"]Check it out![/btn]

DVK-2M computer
Screenshot of the game

I have build my recreation from scratch using my own tetris engine, which supports different colored tetrominos. In it I've recreated the original UI.

How To Play

Controls:

  • 1: show next tetromino
  • 5/space: drop
  • 7 / Left-arrow: Move Left
  • 8 / Up-arrow: Rotate
  • 9 / Right-arrow: Move Right
